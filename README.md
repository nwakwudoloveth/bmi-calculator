## Description
Calculate Body Mass Index with this simple bmi calculator



## Installation
Calculate Body Mass Index with this simple bmi calculator
  
&nbsp;

```
npm install bmi-calculator
```

&nbsp; 
 


## Usage 
  &nbsp;

```
const bmi = require('bmi');
bmi(weight, height); // => (result)
bmi(100, 5.7); // => (3.077870113881194)

```
